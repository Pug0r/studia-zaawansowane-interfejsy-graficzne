W folderze znajduje się plik ze snippetem o nazwie "first-snippet.snippet". 
Aby wykorzystać snippet w Visual Studio należy wykonać kolejno kroki:
	Tools > Code Snippets Manager > Import > wskazać na plik.

Po zaimportowaniu snippetu wystarczy w edytorze wpisać "dfor" i nacisnąć dwukrotnie "tab".
Warto zwrócić uwagę, że kod który snippet generuje się raczej nie skompiluje, trzeba zastąpić "length" i "maxLength" konkretnymi wartościami.
